# Phase 6 — Evaluation Suite

Phase 6 runs the required evaluations and writes a Markdown report.

```bash
source .venv/bin/activate
python phase_6/run_evals.py
```

## Inputs

- `evals/golden_questions.jsonl`
- `evals/adversarial_prompts.jsonl`
- `evals/ux_checks.json`

## Output

- `docs/EVALS_REPORT.md`
