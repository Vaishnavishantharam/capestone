# Phase 2 — Weekly Product Pulse (M2)

This phase mirrors your M2 approach: **reviews CSV → themes (≤5) → top themes (3) → quotes (3) → ≤250 words → 3 action ideas**.

## Run (sample)

```bash
source .venv/bin/activate
python phase_2/run_pulse.py
```

## Run (your CSV)

```bash
source .venv/bin/activate
python phase_2/run_pulse.py --csv data/reviews/YOUR_REVIEWS.csv
```

## Output

- Writes a pulse artifact JSON into `data/state/pulses/`.
